# rom-spork

**ROM Collection Organizer for RetroDECK** — A retro-futuristic desktop application that scans and organizes ROM collections into proper RetroDECK directory structures.

![Retro Terminal](https://via.placeholder.com/800x400/1a1a2e/00ffcc?text=ROM-SPORK+TERMINAL)

## Features

- **Massive Collection Support** — Handles 100,000+ ROM files with optimized threading
- **RetroDECK Integration** — Automatic system detection via file extensions and folder names
- **Cyberpunk Aesthetic** — CRT terminal-inspired UI with neon colors and scanline effects
- **Safety First** — Dry run mode, skip existing files, verify checksums before copying
- **Zero Dependencies Beyond Dear PyGui** — No external assets, icons, or theme folders required
- **Persistent Settings** — Remembers source/destination folders and preferences
- **Comprehensive Logging** — Real-time scrolling log + organizer.log file
- **Report Generation** — CSV and JSON reports of all processed files
